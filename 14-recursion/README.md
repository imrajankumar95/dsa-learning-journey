# Recursion

Breaking problems into smaller subproblems. Base case + recursive case. Foundation for trees, graphs, backtracking, and dynamic programming.

## Key Concepts
- Base case — when to stop
- Recursive case — break problem smaller
- Call stack — each call adds a frame
- Tail recursion — optimization (last operation is recursive call)
- Recurrence relations — T(n) = T(n-1) + O(1)
